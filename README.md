# Монитор курса валют

![Python](https://img.shields.io/badge/python-3.9%2B-blue)
![Dependencies](https://img.shields.io/badge/dependencies-requests-green)

Консольная утилита для отслеживания курсов валют: при каждом запуске
показывает текущие курсы выбранных валют к доллару, сравнивает их
с прошлым запуском (вырос / упал / без изменений) и сохраняет снимок
в локальную историю. Данные берутся из бесплатного API
[open.er-api.com](https://www.exchangerate-api.com) - без ключа и регистрации.

## Возможности

- текущие курсы валют из настраиваемого списка (`WATCHLIST`)
- сравнение с прошлым запуском: направление (`^` / `v` / `=`) и величина изменения
- история всех запусков в `history.json` (время + курсы)
- лёгкая настройка: добавить валюту = дописать её код в список

## Требования

- Python 3.9 или новее
- [requests](https://pypi.org/project/requests/) (единственная зависимость, ставится из `requirements.txt`)
- доступ в интернет

## Установка и запуск

```bash
git clone git@github.com:asadullinalexx-bit/study.git
cd study
pip install -r requirements.txt
python main.py
```

## Использование

Первый запуск - печатает текущие курсы:

```bash
$ python main.py
курс USD на 11.07.2026 19:21
  RUB: 76.3983
  EUR: 0.8756
  CHF: 0.7460
  CNY: 6.7848
история сохранена: 1 запусков
```

Повторные запуски - показывают изменение с прошлого раза:

```bash
$ python main.py
курс USD на 12.07.2026 09:00
  RUB: 76.5100  ^ +0.1117
  EUR: 0.8756  = +0.0000
  CHF: 0.7401  v -0.0059
  CNY: 6.7902  ^ +0.0054
история сохранена: 2 запусков
```

Чтобы следить за другими валютами - отредактируйте список `WATCHLIST`
в начале `main.py` (любые коды валют ISO 4217, например `KZT`, `JPY`):

```python
WATCHLIST = ["RUB", "EUR", "CHF", "CNY"]
```

Чтобы начать историю заново - удалите `history.json`.

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `main.py` | точка входа: печать курсов, сравнение с историей |
| `rates.py` | модуль запроса курсов через open.er-api.com |
| `history.json` | история запусков (создаётся автоматически, в git не хранится) |
| `requirements.txt` | зависимости |
