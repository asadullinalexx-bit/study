# Монитор курса валют

Учебный проект: показывает текущие курсы валют к доллару
и изменение с прошлого запуска.

## Файлы

- `main.py` - точка входа: печатает курсы и ведёт историю
- `rates.py` - запрос курсов через [open.er-api.com](https://www.exchangerate-api.com)
- `history.json` - история запусков (создаётся автоматически, в git не хранится)

## Запуск

```
pip install -r requirements.txt
python main.py
```

Пример вывода:

```
курс USD на 11.07.2026 19:30
  EUR: 0.9214  ^ +0.0003
  RUB: 78.4520  v -0.1200
  GBP: 0.7891  = +0.0000
  CNY: 7.2510
история сохранена: 2 запусков
```
